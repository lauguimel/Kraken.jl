Stream (pull scheme, bounce-back at domain edges)\n        stream_2d!(f_out, f_in, Nx, Ny)\n\n        # 2. BGK collision (in-place on f_out)\n        collide_2d!(f_out, is_solid, ω)\n\n        # 4. Compute macroscopic fields\n        compute_macroscopic_2d!(ρ, ux, uy, f_out)\n\n        # 5. Swap populations\n        f_in, f_out = f_out, f_in\n    end\n\n    # Copy results to CPU\n    ρ_cpu  = Array(ρ)\n    ux_cpu = Array(ux)\n    uy_cpu = Array(uy)\n\n    return (ρ=ρ_cpu, ux=ux_cpu, uy=uy_cpu, config=config)\nend\n```"
