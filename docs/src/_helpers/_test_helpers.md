"```julia\n\"\"\"\n    run_cavity_2d(config::LBMConfig{D2Q9}; backend=CPU(), T=Float64)\n\nRun a 2D lid-driven cavity simulation using D2Q9 BGK-LBM.\n\nReturns a NamedTuple with final fields (ρ, ux, uy) on CPU.\n\"\"\"\nfunction run_cavity_2d(config::LBMConfig{D2Q9};\n                        backend=KernelAbstractions.CPU(), T=Float64)\n    state = initialize_2d(config, T; backend=backend)\n    f_in, f_out = state.f_in, state.f_out\n    ρ, ux, uy = state.ρ, state.ux, state.uy\n    is_solid = state.is_solid\n    Nx, Ny = config.Nx, config.Ny\n    ω = T(omega(config))\n\n    for step in 1:config.max_steps\n        # 1. Stream (pull scheme, bounce-back at domain edges)\n        stream_2d!(f_out, f_in, Nx, Ny)\n\n        # 2. Apply Zou-He velocity BC on lid (post-stream, pre-collision)\n        apply_zou_he_north_2d!(f_out, config.u_lid, Nx, Ny)\n\n        # 3. BGK collision (in-place on f_out)\n        collide_2d!(f_out, is_solid, ω)\n\n        # 4. Compute macroscopic fields\n        compute_macroscopic_2d!(ρ, ux, uy, f_out)\n\n        # 5. Swap populations\n        f_in, f_out = f_out, f_in\n    end\n\n    # Copy results to CPU\n    ρ_cpu  = Array(ρ)\n    ux_cpu = Array(ux)\n    uy_cpu = Array(uy)\n\n    return (ρ=ρ_cpu, ux=ux_cpu, uy=uy_cpu, config=config)\nend\n```"
